# ECS174-FinalProject
Steps to start the project
    1. git clone https://github.com/hadiafifah/ECS174-FinalProject.git
    2. open the folder
    3. source venv/bin/activate
    4. pip install -r requirements.txt
    5. jupyter notebook
    
Steps to prepare the program
    1. go in main branch
    2. create two folders (one for "data" and another for "models")
    3. go in src folder
    4. go in data_processing.ipynb
    5. Run all cells

Steps to run the program
    1. In terminal, cd into src
    2. run "python emotion_model.py
    2. Now run "python webcam.py"

----To close the window Press "esc" or "q" key----

# doing these steps will bring up a camera window that shows the demo of the project
